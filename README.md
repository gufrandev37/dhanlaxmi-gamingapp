# Dhanlaxmi Gaming Admin Panel

A full-featured gaming admin dashboard built using Laravel, designed to manage users, games, bids, wallet transactions, and real-time game results.

---

## 🚀 Features

### 🔐 Admin Panel
- Secure admin authentication
- Role & permission management
- Admin management system

### 🎮 Game Management
- Create & manage games
- Game result declaration
- Game pricing control

### 📊 Bidding System
- Bid history tracking
- Duplicate bid detection
- Real-time bid charts

### 💰 Wallet & Payments
- User wallet management
- Deposit & withdrawal system
- Payment history tracking

### 👥 User Management
- View & manage users
- Wallet history
- Winning history tracking

### 🔔 Notifications
- Send and manage notifications
- Activity tracking

---

## 🛠 Tech Stack

- **Backend:** PHP (Laravel)
- **Database:** MySQL
- **Frontend:** Blade Templates, HTML, CSS, JavaScript
- **Authentication:** Laravel Auth System

---

## 📂 Project Structure

- `app/Http/Controllers/` → Business logic
- `app/Models/` → Database models
- `resources/views/` → Blade templates (UI)
- `routes/` → API & Web routes
- `database/` → Migrations & seeders
- `public/` → Assets (CSS, JS, images)
